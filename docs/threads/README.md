# 线程记录说明

`docs/threads/` 用来存放“单次线程”的交接文件。不是每次都要新建，只有当一次工作背景复杂、容易丢上下文，或者明显会交给下个线程继续时，再创建。

## 命名建议

文件名建议：

`YYYY-MM-DD-主题.md`

例如：

`2026-04-03-context-system-setup.md`

## 什么时候创建

- 一次任务跨多个文件，接手成本高
- 有试验、取舍、失败方案需要留下
- 预计下一线程会直接延续这项工作

## 最低应包含的信息

- 本线程开始时按什么阅读顺序完成了上下文恢复
- 目标
- 改动文件
- 当前结果
- 未完成项
- 风险或注意事项
- 接手建议
